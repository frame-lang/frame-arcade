# Delta inventory — this port vs. Crowther+Woods 1977 CCA

**Purpose:** an honest, categorized list of every place where
this Frame port differs from the canonical 350-point Crowther
& Woods 1977 *Colossal Cave Adventure*. Generated by reading
the source against my knowledge of canon — likely incomplete;
the user will work through it iteratively and add findings.

**Status of EVALUATION.md's "Canon completeness: 100%" claim:**
that referred to the 24-`@@system` count covering every system
the port intended to model, not bit-for-bit prose / scoring
fidelity to the original Fortran source. The deltas below are
real.

**Severity legend:**
- 🔴 **Behavior delta** — the player will observe a different
  outcome from the canonical game (different score, different
  reachable rooms, different puzzle solutions).
- 🟡 **Mechanics delta** — same observable goal but different
  internal mechanism (e.g. vending machine refreshes lamp
  directly vs. dispenses a battery item).
- 🟢 **Prose / flavor delta** — text, room descriptions, or
  responses differ but mechanically equivalent.
- ⚪ **Scope delta** — feature deliberately omitted (the port
  picked a smaller scope; not a bug).

- **Sum of port values:** 5+5+10+10+15+14+14+14+14+14+14+14+14+10+10 = **177**
- **Canon target (15 × 14):** 210

### 2c. Missing fragility

- Canon: vase shatters only when dropped on stone (i.e. anywhere
  except onto the velvet pillow in the egg room). Pillow is the
  recovery mechanism.
- Port: vase shatters on **any** non-deposit drop. No pillow
  recovery path. (Source explicitly: "We simplify".)
